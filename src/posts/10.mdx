# 経緯

2. ディレクトリ構造書いてあげるとわかりやすいかも！
3. `├`とか`─`って何て変換すればいいんだっけ？
4. `tree`使えばええか

# 準備
`tree`コマンドがなかったらinstall
```
$ brew install tree
```

# 実行
```
$ tree -d -L 1
.
├── dir1
├── dir2
└── dir3
```

これをコピペすればおk  
便利だね
