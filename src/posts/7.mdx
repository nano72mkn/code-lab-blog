## TL;DR
`tsconfig.json`のlibに`dom`を追加


## 現象
リポジトリの分割作業で出たエラー。

```
Typescript error: Cannot find name 'Response' 
```

解決策は、`tsconfig.json`のlibに`dom`を追加するだけ

参考：https://github.com/ynab/ynab-sdk-js/issues/96
